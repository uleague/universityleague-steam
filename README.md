# Max Korzh

Max Korzh is a Steam bot for interacting with invited users from uleague.ru

## Installation

1) Clone repository
2) Create .env file
3) Consult for secrets
    - Steam Secrets
    - Backend Secrets
4) Run docker-compose

```bash
docker-compose up --build
```
